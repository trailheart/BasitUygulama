This project built on MacOS.

- Designed and developed a desktop-based registration and authentication system with complex validation rules for credentials.
​Implemented a file-based data persistent model to securely store, read, and manage user accounts with duplication checks.

- ​Developed multi-window desktop management (tk.Toplevel) featuring dynamic visibility toggles, text resizing, and real-time validation error tracking.

- ​Engineered custom event-driven responsive layouts (<Configure>) using pixel-perfect mathematical coordinate computation to keep UI elements centered across massive display resolutions.
